# Support

## How to Get Help

This is a sample/demo repository. Support is provided on a best-effort basis through
GitHub issues.

### Before Opening an Issue

1. Read the [README](README.md) and [SETUP-GUIDE.md](SETUP-GUIDE.md) thoroughly.
2. Check [existing issues](../../issues) (open and closed) — your question may already
   be answered.
3. Review [FINDINGS.md](FINDINGS.md) for known limitations and considerations.

### Opening an Issue

Use the GitHub issue tracker for:

- **Bug reports** — unexpected behavior or errors
- **Setup problems** — issues with EC2 provisioning, Ollama, or SSH tunneling
- **Documentation gaps** — unclear or missing instructions

Please include:
- Your EC2 instance type and AMI
- The Ollama model you are using
- Relevant output from `ec2-setup.sh` or Claude Code
- Steps to reproduce the problem

### Security Issues

Do **not** open a public issue for security vulnerabilities. Follow the process in
[SECURITY.md](SECURITY.md).

### Out of Scope

This repository is sample code and is **not** intended for production use. Issues
requesting production-hardening, SLA commitments, or commercial support are out of scope.
